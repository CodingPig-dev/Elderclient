ElderClient

Use any cape for free, download and manage mods, and view the mods you are currently using — all in one lightweight client.

ElderClient v1.0.0
Lightweight client — single JAR. Click download and run with your Fabric loader.

Features

  • Free cape system with preview
  • Built-in mods repository and downloader
  • Installed mods viewer and manager
  • Simple one-file client site

What ElderClient Does

  • Free cape system and quick switching
  • Integrated mods repository browser and downloader (client handles installs)
  • Shows all currently loaded/installed mods
  • Lightweight, frequently updated client with active development

Capes

ElderClient allows you to use any cape for free. Capes can be previewed inside the client and saved for quick reuse. The client supports both URL-based capes and local images.
